# Pasta para imagens dos banners do carrossel

Esta pasta contém as imagens que serão usadas nos banners do carrossel da Home.

## Estrutura recomendada:

```
public/images/banners/
├── banner-1.jpg (ou .png)
├── banner-2.jpg (ou .png)
├── banner-3.jpg (ou .png)
└── banner-4.jpg (ou .png)
```

## Especificações das imagens:

- **Formato**: JPG ou PNG
- **Dimensões recomendadas**: 244x244px (quadrado)
- **Tamanho**: Máximo 500KB por imagem
- **Qualidade**: Alta resolução para melhor visualização

## Como usar:

1. Adicione suas imagens nesta pasta
2. Renomeie-as para `banner-1.jpg`, `banner-2.jpg`, etc.
3. As imagens serão automaticamente carregadas no carrossel
